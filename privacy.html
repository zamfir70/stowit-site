Privacy Policy

Last updated: February 15, 2026

Summary

StowIt is a local-first application. Your data stays on your device. We do not collect, transmit, or store any personal information.

Data Storage

All data created by StowIt is stored locally on your computer:

  • Drop stack — stored as a JSON file in your local app data directory
  • Session history — stored as local files in your app data directory
  • Configuration — stored as a JSON file in your local app data directory

No data is sent to any server, cloud service, or third party by default.

Network Features

StowIt includes optional network features that are disabled by default:

  • LAN Zones — when enabled, discovers other StowIt instances on your local network using mDNS. Data shared to a zone is transmitted directly between devices on your local network. No data passes through external servers.
  • REST API — when enabled, runs a local HTTP server accessible only on your machine or local network. Protected by a bearer token you control.
  • Integrations — when configured, StowIt can send drop data to services you specify (webhooks, Slack, Discord, Obsidian). Data is sent only to URLs you explicitly configure.

QR Code Transfer

The QR code feature runs a temporary local HTTP server on your device. The QR code points to your local IP address. No data is uploaded to any external service.

Clipboard Access

When clipboard capture is enabled, StowIt monitors your system clipboard to add copied items to your drop stack. This data never leaves your device unless you explicitly share it via Zones or Integrations.

Analytics and Tracking

StowIt does not include any analytics, telemetry, tracking pixels, or third-party SDKs. We do not collect usage data of any kind.

License Verification

Pro license verification is performed entirely offline using cryptographic validation. No network request is made to verify your license.

Third-Party Services

StowIt does not integrate with any third-party services unless you explicitly configure them in Settings. The application makes zero network requests by default.

Children's Privacy

StowIt does not collect any personal information from anyone, including children under 13.

Changes to This Policy

If we update this privacy policy, the changes will be posted on this page with an updated date.
