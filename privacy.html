Privacy Policy

Last updated: February 17, 2026

Summary

StowIt is a local-first application. Your data stays on your device unless you enable Cloud Sync. We do not collect analytics, telemetry, or personal information.

Data Storage

All data created by StowIt is stored locally on your computer:

  • Drop stack — stored as a JSON file in your local app data directory
  • Session history — stored as local files in your app data directory
  • Configuration — stored as a JSON file in your local app data directory

Cloud Sync

StowIt includes an optional Cloud Sync feature for sharing clipboard items across your devices. When enabled:

  • Text, links, and notes are transmitted through a WebSocket relay server to your other devices.
  • Files and images are never transmitted — they stay on your device.
  • The relay server is stateless — it does not store, log, or persist any data. Items are forwarded in real-time and immediately discarded.
  • Connections are grouped by a hash of your license key or sync code. The relay server never sees your actual license key.
  • Item size is capped at 1 MB (Free) or 5 MB (Pro) per item.

Cloud Sync can be disabled at any time in Settings.

Clipboard Access

When clipboard capture is enabled, StowIt monitors your system clipboard to add copied text, links, and notes to your drop stack. You can pause clipboard capture at any time from the system tray menu. Clipboard data never leaves your device unless you explicitly share it via Cloud Sync, Zones, or Integrations.

Network Features

StowIt includes additional optional network features:

  • LAN Zones — when enabled, discovers other StowIt instances on your local network using mDNS. Data shared to a zone is transmitted directly between devices on your local network.
  • REST API — when enabled, runs a local HTTP server accessible only on your machine or local network. Protected by a bearer token you control.
  • Integrations — when configured, StowIt can send drop data to services you specify (webhooks, Slack, Discord, Obsidian). Data is sent only to URLs you explicitly configure.

QR Code Transfer

The QR code feature runs a temporary local HTTP server on your device. The QR code points to your local IP address. No data is uploaded to any external service.

Analytics and Tracking

StowIt does not include any analytics, telemetry, tracking pixels, or third-party SDKs. We do not collect usage data of any kind.

License Verification

Pro license verification is performed entirely offline using cryptographic validation. No network request is made to verify your license.

Children's Privacy

StowIt does not collect any personal information from anyone, including children under 13.

Changes to This Policy

If we update this privacy policy, the changes will be posted on this page with an updated date.
